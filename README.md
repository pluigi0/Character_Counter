# Character Counter

A minimal text analysis tool built with vanilla HTML, CSS, and JavaScript.

## Live Demo

[View it live](https://github.com/pluigi0/Character_Counter)

## Features

- Live character, word, and sentence count
- Estimated reading time
- Optional character limit with progress bar
- Copy to clipboard
- Dark theme

## Tech Stack

HTML · CSS · JavaScript — no libraries, no build tools.

## Run Locally

```bash
git clone https://github.com/pluigi0/Character_Counter.git
cd character-counter
open index.html
```
